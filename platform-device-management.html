摄像头接入指南

进入全局设备管理的摄像头页
全局首页 头像下拉 / 设备管理 / 摄像头
从全局首页右上角头像/设置入口进入
平台级设备管理，并切换到摄像头(GB28181)
新增摄像头并生成接入凭证
点击右上角 添加摄像头
1. 录入设备名称、厂商、型号、安装位置
2. 保存后平台自动生成设备国标编码与接入密码
3. 可在摄像头详情的「国标配置」中查看凭证
登录摄像头厂商后台
通过浏览器访问摄像头 Web 后台
海康参考路径：配置 → 网络 → 高级配置 → 平台接入
大华/宇视路径略有差异，但均需选择 GB28181/国标协议
在设备后台填写 GB28181 注册信息
海康后台对应字段 填入平台生成的凭证
SIP服务器ID SIP 服务器 ID
SIP服务域 (取 SIP 服务器 ID 前 10 位)
SIP服务器地址 / 端口 SIP 服务器 IP / Port
SIP用户名 与 SIP用户认证ID 设备国标编码
SIP用户认证密码 设备接入密码
保存配置，等待设备注册上线
摄像头会主动向 HUB SIP 服务器注册
注册成功后平台状态从「未注册」变为「在线」
同步，设备进入设备池
1. 在摄像头详情或列表中点击「同步」
2. 平台向设备发起目录查询，拉取通道及设备状态
3. 设备留在设备池，所属项目显示为「-」（未分配）
将摄像头分配至具体项目
任选以下一种方式，将设备池中的摄像头领用至目标项目：
方式一：项目编辑（推荐）
全局首页 → 新增项目 / 编辑项目 → 项目设备 → + 添加设备，从池中勾选摄像头
方式二：平台设备管理
设备管理 → 摄像头列表 → 操作「修改项目」，选择目标项目或「暂不分配（留在设备池）」
分配成功后，项目内可在「设备管理-摄像头」与「监控中心」使用该设备
